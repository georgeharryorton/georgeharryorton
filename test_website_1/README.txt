How to edit
- Open any HTML file and replace the placeholder text with your own.
- Update the email in contact.html (mailto:hello@example.com).
- Colours live in style.css under :root (Blue & White theme).

Preview locally
- Double-click index.html to open in your browser.

Deploy for FREE

Option A — GitHub Pages
- Create a GitHub account
- New repository named: yourname.github.io
- Upload all files from this folder (or push via Git)
- Visit https://yourname.github.io after a minute or two

Option B — Netlify (drag & drop)
- Go to app.netlify.com and sign up
- Click "Add new site" → "Deploy manually"
- Drag this whole folder in; it will publish to a *.netlify.app URL

Option C — Vercel
- Import a GitHub repo with these files; select "Other" framework
- Deploy; you'll get a *.vercel.app URL
